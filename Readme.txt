Öppna program mappen i valfri IDE
Välj Project Mappen (Mortage)
Kör main.java
Programmet kommer då öppna prospects.txt i "src/main/resources" och läsa in varje användare rad för rad.
Sedan kommer Mortage att räknas ut för varje användare(Detta görs när man varje användare läses in från filen) och resultatet
kommer skrivas ut i output fönstret.
